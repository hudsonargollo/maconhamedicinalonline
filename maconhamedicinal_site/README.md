
# maconhamedicinal.online

Projeto de exemplo em **Next.js** para a landing page da plataforma de acompanhamento com cannabis medicinal.

## Como usar

1. Instale as dependências:

```bash
npm install
```

2. Rode o servidor de desenvolvimento:

```bash
npm run dev
```

3. Acesse em: http://localhost:3000

> Obs: Este é um template inicial de identidade visual e landing page. Ajuste textos, cores e seções conforme evolução do projeto real.
